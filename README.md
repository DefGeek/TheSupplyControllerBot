# TheSupplyControllerBot

**TheSupplyControllerBot** — Telegram‑бот, предназначенный для автоматизации взаимодействия в темах "заявки" и "доставки" с созданием таблиц Google Sheets и регистрацией пользователей.

---

## ⚙️ Основной функционал
- При добавлении в группу бот создаёт новую Google‑таблицу.
- Название листа формируется из наименования и кода объекта.
- По команде `/start` бот регистрирует пользователя (ФИО, должность, телефон) и сохраняет данные в базу SQLite.
- При последующих обращениях фамилия и инициалы подставляются автоматически.
- Поддерживаются только темы "заявки" и "доставки".
- Все конфиденциальные данные хранятся в `.env`.

---

## 📦 Установка зависимостей
```bash
cd TheSupplyControllerBot
pip install -r requirements.txt
```

---

## 🔐 Настройка окружения
Создайте файл `.env` на основе `.env.example`:
```bash
cp .env.example .env
```
и заполните своими значениями:
```
BOT_TOKEN=ваш_токен_тг_бота
GOOGLE_CREDENTIALS=credentials.json
GOOGLE_SCOPES=https://www.googleapis.com/auth/spreadsheets
```

Поместите в папку файл `credentials.json`, полученный из Google Cloud Console.  
Не добавляйте его в Git — он в `.gitignore`.

---

## ▶️ Запуск
```bash
python3 main.py
```

При запуске:
- Создаётся база данных `users.db` (если её нет).
- Бот начинает принимать команды `/start` и `/help`.
- При добавлении в группу бот предложит ввести наименование и код объекта и создаст Google Spreadsheet.

---

## 📘 Доступные команды
| Команда | Описание |
|----------|-----------|
| `/start` | Регистрация пользователя |
| `/help` | Список доступных команд |
| _Добавление бота в группу_ | Создание Google Sheet для объекта |
